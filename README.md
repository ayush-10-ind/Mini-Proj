# Mini-Proj
An Explainable AI for Privacy Preserving and collaborative Diagnostic Platform. This is part of our college Mini project and as of now we have implemented only the explainable part of it and we will expand this further in the upcoming semesters. This video is showcasing the implementation of phase - 1 i.e., The explainable part by one of my team member.

OUR VISION :- A centralized Dashboard that manages training and visualizes explanation from remote client in real time.

FEATURES it will have :- 
[i] Live Fedration Monitor (A central screen showing the status of connected hospitals and a graph of Global Model Accuracy)
[ii] Interactive XAI Module
[iii] DRIFT DETECTION ALERT SYSTEM

KEY TECHNOLGIES USED:- 
RESNET-18 (A Pre-trained Model)
GRAD-CAM
DATASET (from KAGGEL) [https://www.kaggle.com/datasets/paultimothymooney/chest-xray-pneumonia]
